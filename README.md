## English

### Overview

This is an OpenClaw skill providing comprehensive knowledge of the 十八占 (18 Divinations), the classical Chinese divination system from the I Ching tradition. Each of the 18 methods serves a different purpose and situation.

### Features

- **18 Distinct Methods** - Each divination type has unique application
- **Traditional Foundation** - Based on I Ching (易经) methods
- **Practical Guide** - How and when to use each method
- **Step-by-Step Process** - From preparation to interpretation

### Installation

1. Copy `shi-ba-zhan` folder to your OpenClaw skills directory:
   - **Windows**: `%USERPROFILE%\.qclaw\workspace\skills\`
   - **macOS/Linux**: `~/.qclaw/workspace/skills/`
2. Restart OpenClaw

### Usage

This skill activates when asking about:
- 十八占 / 18 Divinations
- Chinese fortune telling
- Oracle reading methods
- I Ching divination
- 易经占卜

## 中文

### 概述

这是一套基于《易经》传统的十八种占卜方法体系。每种方法针对不同的情况和用途。

### 功能

- **18种方法** - 每种占法各有其用
- **传统根基** - 基于易经体系
- **实用指南** - 何时使用何种方法
- **步骤详解** - 从准备到解卦

### 安装

将 `shi-ba-zhan` 文件夹复制到 OpenClaw 技能目录：
- **Windows**: `%USERPROFILE%\.qclaw\workspace\skills\`
- **macOS/Linux**: `~/.qclaw/workspace/skills/`

### 使用场景

- 十八占 / 18种占卜
- 中国算命
- 易经占法
- 问卦算卦

### 核心方法

1. **静占** - 心神宁静时占问
2. **悔占** - 后悔改正时占问
3. **专占** - 专心一事时占问
4. **享占** - 亨通顺利时占问
5. **征占** - 战争竞争时占问
6. **师占** - 军队团体时占问
7. **谷占** - 农业收成时占问
8. **史占** - 借鉴历史时占问
9. **辞占** - 解读卦辞时占问
10. **象占** - 观察象数时占问
11. **策占** - 计划策略时占问
12. **姓占** - 姓氏相关时占问
13. **卜占** - 传统蓍草法
14. **梦占** - 梦境预示时占问
15. **失占** - 寻找失物时占问
16. **药占** - 疾病健康时占问
17. **产占** - 生产生育时占问
18. **行人占** - 出行归期时占问

### 占卜流程

1. **净手焚香** - 清洁准备
2. **默祷问题** - 心中发问
3. **掷币起卦** - 三币六次
4. **读卦解卦** - 分析含义
